# JodyCraft Portfolio

**AI × C# で業務を変える** — 製造業エンジニアのポートフォリオサイト

## 概要

製造業の生産技術部門で培った C# 開発経験を活かし、業務自動化・DX支援を行う JodyCraft のポートフォリオサイトです。

## 技術スタック

- **Blazor WebAssembly** (.NET 9)
- **Bootstrap 5** (CSS Framework)
- **Chart.js** (グラフ描画)
- **GitHub Pages** (ホスティング)
- **GitHub Actions** (CI/CD)

## ローカル開発

```bash
# プロジェクトのビルド
dotnet build

# ホットリロード付きで起動
cd src/JodyCraft.Portfolio
dotnet watch run
```

ブラウザで `https://localhost:5001` にアクセスしてください。

## デプロイ

`main` ブランチへの push で GitHub Actions が自動起動し、GitHub Pages にデプロイされます。
